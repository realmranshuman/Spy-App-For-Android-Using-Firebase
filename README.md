# Spy-App-For-Android-Using-Firebase
### SMS And Call Logs Backup to Firebase Realtime Database

This Android project demonstrates how to backup SMS messages to Firebase Realtime Database. It includes functionality to fetch SMS messages from the device and store them in Firebase.

## Prerequisites

- Android Studio 
- Firebase account and project
- Android device or emulator with SMS messages

## Setup

1. Clone the repository to your local machine or download the ZIP file.
2. Open Android Studio and select "Open an Existing Project".
3. Navigate to the project folder and select it.
4. Wait for the project to load and sync with Gradle.

## Firebase Configuration

1. Create a Firebase account and project on the [Firebase Console](https://console.firebase.google.com/).
2. Add an Android app to your Firebase project and follow the instructions to generate a `google-services.json` file.
3. Place the `google-services.json` file in the `app/` directory of the project.
